# 🧠 AI SYSTEM INSTRUCTIONS FOR SENTIRIC-PLAYGROUND

1. **SDK Otoritesi:** Bu repo içinde ASLA `@sentiric/stream-sdk` kütüphanesini `node_modules` üzerinden doğrudan import etme. Daima `src/lib/sdk-provider.ts` köprüsünü (Bridge) kullan.
2. **Yasak:** Çözümlerin (`src/solutions/`) içine iş mantığı (AI promptları, backend routing kararları) yazma. Bu işler `dialog-service`'e aittir.
3. **Modülerlik:** Bir çözüm geliştirirken asla dış klasördeki bir CSS veya JS dosyasını kirletme. Her şey kendi klasöründe (Self-Contained) kalmalıdır.
4. **Pretty URLs:** Link verirken asla `.html` uzantısı kullanma. Dizin bazlı `/solutions/name/` yapısını koru.
